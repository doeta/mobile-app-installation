Instalasi Kotlin Multiplatform (KMM)

Platform: Multiplatform (Kotlin) • Waktu: ~40 Menit

1 Prasyarat

Pastikan Anda sudah menginstal:

  • Android Studio (versi terbaru)
  • XCode (jika menggunakan macOS untuk target iOS)
  • JDK 17 atau lebih baru

2 Install Plugin KMP

Buka Android Studio, masuk ke Settings/Preferences > Plugins. Cari dan install plugin Kotlin Multiplatform (sebelumnya bernama Kotlin Multiplatform Mobile).

3 KDoctor

Untuk memastikan lingkungan siap, disarankan menggunakan tool command line kdoctor (hanya macOS). Jalankan di terminal: 

brew install kdoctor
kdoctor

Perbaiki masalah yang dilaporkan oleh kdoctor.

4 Membuat Project Baru

Di Android Studio, pilih New Project. Di panel kiri, pilih Kotlin Multiplatform. Pilih template Kotlin Multiplatform App.

5 Konfigurasi Project

Beri nama project HelloWorldKMM. Pastikan opsi untuk iOS framework distribution dipilih (biasanya Regular framework atau CocoaPods).

6 Struktur Project & Run

Project akan memiliki folder androidApp, iosApp, dan shared. Logika bisnis ditulis di shared.

Pilih konfigurasi run androidApp untuk menjalankan di Android Emulator. Untuk iOS, buka iosApp.xcodeproj di XCode atau pilih konfigurasi iosApp di Android Studio (jika plugin mendukung). 

// shared/src/commonMain/kotlin/Greeting.kt
class Greeting {
    private val platform: Platform = getPlatform()

    fun greet(): String {
        return "Hello, ${platform.name}!"
    }
}
