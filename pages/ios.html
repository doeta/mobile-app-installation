Instalasi XCode & Project HelloWorld

Platform: iOS (Swift) • Waktu: ~45 Menit • Syarat: macOS

1 Download XCode

Buka App Store di Mac Anda dan cari "Xcode". Klik tombol Get atau Install. Ukurannya cukup besar (lebih dari 10GB), jadi pastikan koneksi internet Anda stabil.

2 Instalasi Komponen Tambahan

Setelah selesai diunduh, buka XCode. Saat pertama kali dibuka, XCode akan meminta izin untuk menginstal komponen tambahan. Masukkan password Mac Anda dan biarkan proses selesai.

3 Membuat Project Baru

Di layar Welcome to Xcode, pilih Create a new Xcode project.

4 Memilih Template

Pilih tab iOS, lalu pilih App di bagian Application. Klik Next.

5 Konfigurasi Project

Isi detail project:

  • Product Name: HelloWorld
  • Organization Identifier: com.example
  • Interface: SwiftUI (atau Storyboard)
  • Language: Swift

6 Menjalankan Aplikasi

XCode akan membuka editor. Di file ContentView.swift, Anda akan melihat kode SwiftUI default. Pilih Simulator (misal iPhone 15) di bagian atas, lalu klik tombol Play. 

import SwiftUI

struct ContentView: View {
    var body: some View {
        VStack {
            Image(systemName: "globe")
                .imageScale(.large)
                .foregroundStyle(.tint)
            Text("Hello, world!")
        }
        .padding()
    }
}
