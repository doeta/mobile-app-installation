Instalasi React Native & Project HelloWorld

Platform: Multiplatform (JS/TS) • Waktu: ~20 Menit

1 Instalasi Node.js

React Native membutuhkan Node.js. Unduh versi LTS dari nodejs.org dan install.

2 Setup Lingkungan (React Native CLI)

Untuk pengembangan native penuh, kita akan menggunakan React Native CLI. Anda perlu menginstal JDK (Java Development Kit) dan Android Studio (lihat tutorial Android). Pastikan variabel environment ANDROID_HOME sudah diset.

3 Membuat Project Baru

Buka terminal atau command prompt. Jalankan perintah berikut untuk membuat project baru:

npx react-native@latest init HelloWorld

Ketik y jika diminta untuk menginstal paket create-react-native-app.

4 Menjalankan Metro Bundler

Masuk ke direktori project:

cd HelloWorld
npm start

Ini akan menjalankan Metro Bundler yang bertugas mem-bundle JavaScript code.

5 Menjalankan di Android

Buka terminal baru (biarkan Metro berjalan). Pastikan emulator Android sudah berjalan atau device terhubung. Jalankan:

npm run android

6 Edit App.tsx

Buka folder project di VS Code. Edit file App.tsx untuk mengubah teks menjadi "Hello World".

import React from 'react';
import { Text, View } from 'react-native';

const App = () => {
  return (
    
      Hello, World!
    
  );
}
export default App;
