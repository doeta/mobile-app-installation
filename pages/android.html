Instalasi Android Studio & Project HelloWorld

Platform: Android (Kotlin) • Waktu: ~30 Menit

Persiapan Sebelum Instalasi

Sebelum kita membahas cara install Android Studio di Windows atau sistem operasi lainnya, ada beberapa hal yang harus kamu siapkan:

1. Spesifikasi Minimum Laptop/PC

Agar Android Studio bisa berjalan dengan lancar, berikut adalah spesifikasi minimum yang disarankan:

  • Sistem operasi: Windows 10/11 (64-bit), macOS (Chip Intel/Apple Silicon), atau Linux.
  • RAM: Minimal 8 GB (lebih disarankan 16 GB).
  • Penyimpanan: Minimal tersedia 8 GB (lebih disarankan SSD).
  • Resolusi layar: Minimum 1280 x 800.

2. JDK (Java Development Kit)

Android Studio membutuhkan Java untuk menjalankan environment-nya. Namun sejak beberapa versi terakhir, JDK sudah disertakan otomatis saat instalasi, jadi kamu tidak perlu menginstal secara terpisah.

3. Koneksi Internet Stabil

Selama proses instalasi dan konfigurasi SDK, Android Studio akan mengunduh berbagai komponen penting. Pastikan kamu memiliki koneksi internet yang stabil agar proses berjalan lancar.

1 Unduh Installer

Kunjungi situs resmi Android Studio di developer.android.com/studio. Klik tombol Download Android Studio dan pilih versi untuk Windows. Setujui lisensi, lalu simpan file .exe di komputer kamu.

2 Jalankan Installer

Setelah proses unduh selesai, klik dua kali file installer tersebut. Akan muncul wizard instalasi Android Studio.

3 Ikuti Panduan Instalasi

Klik Next hingga kamu masuk ke bagian pemilihan komponen. Biarkan opsi default tercentang (Android Studio dan Android Virtual Device), lalu klik Next dan “Install”.

4 Tunggu Hingga Instalasi Selesai

Proses ini bisa memakan waktu beberapa menit tergantung spesifikasi laptop kamu.

5 Buka Android Studio

Setelah instalasi selesai, klik Finish untuk membuka Android Studio. Saat pertama kali dibuka, kamu akan diminta mengatur beberapa konfigurasi awal.

6 Setup SDK dan Emulator

Wizard konfigurasi akan muncul, pilih Standard untuk settingan default. Kemudian Android Studio akan mengunduh SDK, platform tools, dan emulator.

7 Selesai!

Setelah semuanya selesai, kamu akan dibawa ke tampilan utama Android Studio dan bisa mulai membuat project pertamamu.

8 Membuat Project Baru

Klik "New Project". Pilih template "Empty Views Activity" (untuk XML) atau "Empty Activity" (untuk Jetpack Compose). Untuk tutorial ini kita gunakan Empty Activity.

9 Konfigurasi Project

Isi detail project sebagai berikut:

  • Name: HelloWorld
  • Package name: com.example.helloworld
  • Language: Kotlin
  • Minimum SDK: API 24 (Android 7.0)

10 Menjalankan Aplikasi

Tunggu proses Gradle sync selesai. Buka file MainActivity.kt. Anda akan melihat kode dasar "Hello Android". Klik tombol Run (segitiga hijau) di toolbar atas untuk menjalankan di Emulator. 

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContent {
            HelloWorldTheme {
                Surface(modifier = Modifier.fillMaxSize(), color = MaterialTheme.colorScheme.background) {
                    Greeting("World")
                }
            }
        }
    }
}
