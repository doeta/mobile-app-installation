Instalasi Android Studio & Project HelloWorld

Platform: Android (Kotlin) • Waktu: ~30 Menit

1 Download Android Studio

Kunjungi situs resmi developer.android.com/studio dan unduh versi terbaru Android Studio (saat ini Ladybug atau Koala).

2 Proses Instalasi

Jalankan installer yang telah diunduh. Pastikan untuk mencentang "Android Virtual Device" saat diminta komponen apa yang ingin diinstall.

3 Konfigurasi SDK

Setelah instalasi selesai, buka Android Studio. Setup Wizard akan berjalan untuk mengunduh Android SDK terbaru. Pastikan koneksi internet stabil.

4 Membuat Project Baru

Klik "New Project". Pilih template "Empty Views Activity" (untuk XML) atau "Empty Activity" (untuk Jetpack Compose). Untuk tutorial ini kita gunakan Empty Activity.

5 Konfigurasi Project

Isi detail project sebagai berikut:

  • Name: HelloWorld
  • Package name: com.example.helloworld
  • Language: Kotlin
  • Minimum SDK: API 24 (Android 7.0)

6 Menjalankan Aplikasi

Tunggu proses Gradle sync selesai. Buka file MainActivity.kt. Anda akan melihat kode dasar "Hello Android". Klik tombol Run (segitiga hijau) di toolbar atas untuk menjalankan di Emulator. 

class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContent {
            HelloWorldTheme {
                Surface(modifier = Modifier.fillMaxSize(), color = MaterialTheme.colorScheme.background) {
                    Greeting("World")
                }
            }
        }
    }
}
