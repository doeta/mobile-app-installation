Instalasi Flutter & Project HelloWorld

Platform: Multiplatform (Dart) • Waktu: ~30 Menit

1 Download Flutter SDK

Unduh Flutter SDK dari situs resmi flutter.dev. Ekstrak file zip ke lokasi yang diinginkan (misal: C:\src\flutter).

2 Update Path

Tambahkan folder flutter/bin ke Environment Variable PATH sistem Anda agar perintah flutter bisa dijalankan dari mana saja.

3 Flutter Doctor

Buka terminal dan jalankan perintah:

flutter doctor

Perintah ini akan mengecek dependensi yang kurang. Ikuti instruksi yang muncul (misal: install Android Studio, setujui lisensi Android SDK).

4 Install Plugin VS Code

Buka VS Code, masuk ke tab Extensions, cari dan install ekstensi Flutter (ini akan otomatis menginstall ekstensi Dart).

5 Membuat Project Baru

Di VS Code, tekan Ctrl+Shift+P (Command Palette), ketik "Flutter: New Project", pilih "Application", lalu pilih folder penyimpanan dan beri nama project helloworld.

6 Menjalankan Aplikasi

Buka lib/main.dart. Ubah kode sesuai kebutuhan atau biarkan default counter app. Pilih device di status bar kanan bawah, lalu tekan F5 untuk debug. 

import 'package:flutter/material.dart';

void main() {
  runApp(const MainApp());
}

class MainApp extends StatelessWidget {
  const MainApp({super.key});

  @override
  Widget build(BuildContext context) {
    return const MaterialApp(
      home: Scaffold(
        body: Center(
          child: Text('Hello World!'),
        ),
      ),
    );
  }
}
