# Mobile Development Setup Guide

Website ini berisi panduan instalasi untuk berbagai lingkungan pengembangan aplikasi mobile.

## Struktur Folder

- `index.html`: Halaman utama.
- `style.css`: File CSS untuk styling.
- `script.js`: File JavaScript untuk interaksi (menu mobile).
- `pages/`: Berisi halaman tutorial untuk setiap platform.
- `images/`: Tempat menyimpan screenshot tutorial.
